## Getting started

Clone this project

```
git clone https://github.com/MatheusNadai/gofinances-app-rn.git
```

Install dependencies with npm or yarn, choose one

```
yarn
```

```
npm -i
```

#### Starting your project

Android

```
yarn android
```

iOS

```
yarn ios
```

That's all :)
